# AI Agent Skill

## Scope

- LLM 接入
- LangChain
- Tool Calling
- MCP
- Memory
- Skill 设计
- Workflow 编排
- Bot / Assistant 系统

## Agent Thinking Model

遇到 Agent 问题时，优先拆成：

- model
- prompt
- tool
- memory
- workflow
- channel
- observability

## Default Preferences

- 能工具化就不要全部堆在 prompt
- 能模块化就不要做成黑盒
- 能调试就不要只追求“看起来聪明”
- 能复用就不要只做一次性 agent

## Delivery Preference

优先输出：
- 系统结构图
- 目录结构
- 模块职责
- 最小实现
- 后续扩展方式
