# Product Skill

## Scope

- Web 产品包装
- README 编写
- 一键部署
- 上手流程设计
- 体验与交付优化

## Product Sense

产品不只是“功能存在”，还包括：
- 用户看不看得懂
- 配置麻不麻烦
- 启动方不方便
- 是否能快速体验
- 是否知道出错后怎么处理

## Preferred Delivery

优先考虑：
- 在线演示
- 一键部署
- 快速开始
- 截图 / 架构图
- 明确文档

## Good Product Output

- 新用户能快速跑起来
- 老用户能快速维护
- 读 README 就知道项目价值
