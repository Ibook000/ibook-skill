# 嘉立创 MSPM0G3507 开发板详细引脚映射文档

本文档基于立创开发板（MSPM0G3507）的原理图与引脚图汇总，一一对照了各个排针接口、MCU原生引脚以及板载外设的连线关系。

## 2. 板载外设接口对照

### 2.1 SPI Flash (U24)
| Flash 模块引脚 | MCU 引脚 | 对应外设复用功能 |
| :--- | :--- | :--- |
| W_CS (片选) | PB6 | SPI1_CS0 (引脚 58) |
| W_MISO (数据输出) | PB7 | SPI1_POCI (引脚 59) |
| W_MOSI (数据输入) | PB8 | SPI1_PICO (引脚 60) |
| W_CLK (时钟) | PB9 | SPI1_SCK (引脚 61) |

### 2.2 SPI OLED 屏幕接口 (支持 0.96/1.3寸屏)
| OLED 排母丝印 | MCU 引脚 | 功能说明 |
| :--- | :--- | :--- |
| LCD_SCL (时钟) | PB9 | 硬件 SPI1_SCK |
| LCD_SDA (数据) | PB8 | 硬件 SPI1_PICO |
| LCD_RES (复位) | PB10 | 普通 GPIO |
| LCD_DC (数据/命令) | PB11 | 普通 GPIO |
| LCD_CS (片选) | PB14 | 普通 GPIO |
| BLK (背光控制) | PB26 | 普通 GPIO |

### 2.3 CH340E 串口与下载 (U18)
| CH340E 转换引脚 | MCU 原生引脚 | 作用与功能说明 |
| :--- | :--- | :--- |
| TXD (340_TX) | PA11 (U0RX / BSLRX) | 用于接收 PC 端串口数据及 BSL 烧录指令 |
| RXD (340_RX) | PA10 (U0TX / BSLTX) | 用于向 PC 端发送串口数据 |

### 2.4 SWD 调试接口 (U20)
此接口同时具备 SWD 调试和串口直连功能。
| SWD 接口丝印 | MCU 对应引脚 / 供电说明 |
| :--- | :--- |
| PA10/U0TX | PA10 |
| PA11/U0RX | PA11 |
| SWCLK | PA20 (SWD 时钟信号) |
| SWDIO | PA19 (SWD 数据信号) |
| VBUS_5V | USB 提供 5V 电源输出 |
| EXT_3V3 | 降压后提供的 3.3V 系统电源 |
| GND | 系统地 |

## 3. MCU 核心引脚 (U1 MSPM0G3507-64P) 分类总览

### 3.1 模拟与 ADC 通道
- **PA15 ~ PA21**: 对应 ADC 通道 A1_0 到 A1_7。
- **PA22 ~ PA27, PB20, PB24, PB25**: 对应 ADC 通道 A0_0 到 A0_7。
- **VREF / 外部参考**: PA21 (VREF-), PA23 (VREF+)。

### 3.2 通讯总线接口
- **CAN 总线**: PA12 (CAN_TX), PA13 (CAN_RX) / PA26 (CAN_TX), PA27 (CAN_RX)。
- **默认烧录与调试串口**: PA10 (U0TX) 和 PA11 (U0RX)。

### 3.3 时钟与晶振
- **外部低频时钟 (LFX)**: PA3 (LFXIN), PA4 (LFXOUT)。
- **外部高频时钟 (HFX)**: PA5 (HFXIN), PA6 (HFXOUT)。
- **内部阻容时钟参考**: PA2 (ROSC)。
