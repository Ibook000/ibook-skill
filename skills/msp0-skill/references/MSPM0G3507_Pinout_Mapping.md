# 立创·天猛星 MSPM0G3507 V1.0.2 排针映射

本文档依据用户提供的《天猛星MSPM0G3507开发板引脚图.pdf》V1.0.2 开发板正面图重新核对。

## 使用范围与重要原则

- 仅适用于图片所示的 **LCKFB 天猛星 MSPM0G3507 V1.0.2**。
- 观察方向：开发板正面朝上，板上文字可正常阅读，黄色 `5V TX RX GND` 四针接口位于底部。
- 左列、右列均按图片中的视觉方向描述。
- 每侧都是 2×20 排针；行号从最顶部的 `GND/GND` 开始向下数。
- 该引脚图没有给出可安全套用的 U21/U22 连接器针号。不要根据其他开发板、其他版本或未经核对的原理图推导 U21/U22 编号。
- 实际接线优先认开发板丝印 `A00/A01/B14/B11/CLK/DIO/RST`，SysConfig 中对应写作 `PA0/PA1/PB14/PB11/PA20/PA19/NRST`。
- 如果开发板版本不是 V1.0.2，必须重新核对对应版本的 PDF 或原理图。

## 4. 特殊与系统相关引脚

天猛星资料把以下引脚标为特殊引脚或要求谨慎使用：

- PA21 / A21
- PA23 / A23
- PA2 / A02
- PA18 / A18
- PA10 / A10
- PA11 / A11

使用规则：

- PA18 是 BSL 相关引脚。复位时被外部拉到错误电平可能进入 BSL，而不是运行用户程序。
- PA10/PA11 常用于 UART0、串口下载或外接串口模块。项目已经明确使用时可以保留，但不要随意改作普通 GPIO。
- PA21/PA23 与模拟参考功能相关，PA2 与系统/ROSC 相关；普通外设优先选其他引脚。
- PA19/PA20 分别是 SWDIO/SWCLK，调试时不得与其他设备争用。
- PA5/PA6 在使用板载 40 MHz HFXT 的工程中属于晶振引脚，不应分配给普通外设；它们未出现在该 V1.0.2 两侧排针表中。

## 5. 已能从 V1.0.2 图片确认的板级资源

### PB22 LED

图片在 B22 附近标出了 LED，因此 PB22 可作为板载 LED 引脚。已验证工程中常见逻辑为低电平点亮、高电平熄灭，但实际极性仍应以对应板版本原理图和实测为准。

### SWD 调试

- `CLK`：PA20 / SWCLK，右侧第 10 行左列
- `DIO`：PA19 / SWDIO，右侧第 10 行右列
- `RST`：NRST，左侧第 18 行右列
- 调试器还必须连接 GND 和 3.3V 目标电压参考

### 底部四针串口/供电接口

V1.0.2 图片底部标出了 `5V TX RX GND` 四针接口。仅凭引脚图不能安全证明 TX/RX 与哪组 MCU UART 的完整板内连接，也不能据此断言板上集成了 CH340。

- 若本地原理图或经过实测的工程确认它连接 UART0 PA10/PA11，可按该工程使用。
- 若缺少原理图或实测，不要把“板载 CH340”作为所有天猛星版本的固定事实。
- 外接 USB-TTL 时使用 3.3V TTL 电平，并交叉连接 TX/RX。

## 6. 不应再使用的旧假设

以下内容不适用于这份 V1.0.2 正面图，禁止继续作为默认事实：

- 把两侧排针直接命名为 U21/U22，并使用旧的 1～40 针编号。
- 声称 PA0 位于“U21-3”、PA1 位于“U21-5”等未经该板 PDF 证明的编号。
- 声称 PB11/PB14 因板载 OLED 而固定占用；V1.0.2 引脚图只证明它们被引出为 B11/B14，是否连接额外模块必须查对应原理图。
- 声称所有天猛星 MSPM0G3507 版本都板载 CH340；V1.0.2 图片只明确显示底部 `5V TX RX GND` 接口。
- 将 MSPM0G3519 自定义板的 OLED、IMU、Flash、无线、WS2812、蜂鸣器占用表套用到天猛星 MSPM0G3507。

## 7. Agent 使用要求

当 Agent 为天猛星 MSPM0G3507 提供接线建议时：

1. 先确认开发板版本。
2. V1.0.2 使用本文档的正面视觉行列和丝印映射。
3. 优先输出 `PAxx/PBxx + 板上丝印`，行列位置仅作为辅助。
4. 不得自行补出 U21/U22 针号。
5. SysConfig 生成头文件负责确认 MCU 复用功能；开发板 PDF/原理图负责确认物理排针位置，两者缺一不可。
6. 对板载外设、上拉电阻、串口芯片和电源连接，只报告 PDF/原理图或实测能够证明的内容。
