# Coding Skill

## Scope

- Python
- 自动化脚本
- API 封装
- 模块化重构
- 调试排错
- 工程实战代码

## Coding Preference

- 先写能跑版本
- 再整理结构
- 默认给完整函数或完整模块
- 默认包含必要注释
- 默认加基础错误处理
- 默认说明改动点

## What good output looks like

- 可以直接复制运行
- 输入输出明确
- 依赖说明清楚
- 关键逻辑不藏着
- 不是只给片段

## Anti-Patterns

避免：
- 只给思路不写代码
- 只改局部却不说明上下文
- 代码跑不起来
- 把简单逻辑绕复杂
