# Trading Skill

## Scope

- 策略逻辑
- K线连续判断
- 状态机
- 自动下单
- 多市场切换
- 风控与执行约束
- 重试、重连、时间锁

## Core Trading Philosophy

- 策略优先
- 风控优先
- 执行纪律优先
- 情绪不能直接接管交易动作

## Required Structure

任何交易方案必须尽量写清楚：

- 开仓条件
- 加仓条件
- 补单条件
- 止损条件
- 止盈条件
- 退出条件
- 时间锁
- 防重复下单
- 异常处理

## Common Reflection Bias

注意这类典型错误：
- 方向判断对了但拿不住
- 踏空后追单
- 重仓赌方向
- 浮盈靠情绪平仓
- 震荡行情频繁操作
- 没有把规则写成系统约束

## Ideal Output

交易相关输出应尽量具备：
- 明确条件
- 明确状态
- 明确时间点
- 明确风控
- 明确代码实现路径
