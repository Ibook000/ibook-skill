# SVG Chart Snippets for Tweet Card Generator All charts are inline SVG — no external dependencies. They render perfectly in headless Chromium. Copy the relevant snippet into the `.chart-container` div in `card-template.html`. ## 1. Horizontal Bar Chart (4-6 items) Best for: rankings, comparisons, market share breakdown. ```svg ``` ## 2. Vertical Bar Chart (5-8 items) Best for: time series, quarterly data, category comparison. ```svg ``` ## 3. Sparkline (minimal line, 1 metric trend) Best for: price trends, growth curves, showing momentum. ```svg ``` ## 4. Donut / Percentage Ring Best for: market share, percentage metrics, allocation. ```svg ``` ## 5. Mini Comparison (two side-by-side blocks) Best for: A vs B, before/after, this year vs last year. ```svg ``` ## 6. Grouped Bars（两组对比柱状图） Best for: sector vs sector, before vs after, this year vs last. ```svg ``` ## 7. Waterfall（增减瀑布图） Best for: showing how a value changed step by step (revenue bridge, cost breakdown). ```svg ``` ## 8. Progress / Gauge Row (3 items) Best for: adoption metrics, completion rates, progress indicators. ```svg ``` ## 9. Pie Chart（饼图） Best for: composition, market share, portfolio allocation. SVG pie via circle dasharray. ```svg ``` ## 10. Flow Diagram（流程图 / 资金链路图） Best for: supply chains, money flows, process steps, organizational structure. ```svg ``` ## 11. Stacked Bar（堆叠柱状图） Best for: showing composition within total, part-to-whole across categories. ```svg ``` ## 12. Area Chart（面积图） Best for: cumulative trends, volume over time, sentiment shift. ```svg ``` --- ## Quick Reference: Which Chart When | 数据类型 | 推荐图表 | 编号 | |----------|----------|:--:| | 排名 / 份额 | Horizontal Bars | #1 | | 时间序列 | Vertical Bars / Area | #2 / #12 | | 趋势线 | Sparkline / Area | #3 / #12 | | 百分比 | Donut Ring / Pie | #4 / #9 | | A vs B 对比 | Mini Comparison / Grouped Bars | #5 / #6 | | 进度 / 达成率 | Progress Bars | #8 | | 增减分解 | Waterfall | #7 | | 构成 / 组成 | Stacked Bar / Pie | #11 / #9 | | 流程 / 链路 | Flow Diagram | #10 | | 多组对比 | Grouped Bars | #6 | ## 12.
