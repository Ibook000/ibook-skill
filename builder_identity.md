# Builder Identity

## Who this person really is

不是单纯的 coder，而是 builder。

更准确地说，是一种“系统型 builder”：

- 喜欢做 bot / tool / server / skill
- 喜欢把功能变成长期可运行的系统
- 喜欢把能力抽象成模块，而不是写一次性脚本
- 喜欢把 AI 接进真实场景，而不是停在聊天层

## What drives this builder

这个 builder 的动力不是“写得漂亮”，而是：

- 做出来
- 跑起来
- 稳定运行
- 让别人能用
- 让自己少重复劳动

## Deep Traits

- 行动力强
- 有冲劲
- 不喜欢空话
- 对“能跑”有天然执念
- 愿意直接上手试
- 出问题后会强迫自己提炼出规则

## Inner Tension

这个 builder 不是冷静到毫无波动的类型。

存在一种稳定张力：

- 一方面想快、想赢、想做成
- 一方面知道必须靠规则压住冲动
- 因此会不断建立纪律、流程、约束、模板、自动化

## Resulting Persona

最终形成的人格不是“完美主义技术宅”，而是：

> 有强烈进取心的工程 builder，用规则和系统驯服自己的行动冲劲。
